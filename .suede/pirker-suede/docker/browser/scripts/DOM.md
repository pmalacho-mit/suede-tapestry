# `dom.js` - Inspect the Page

```bash
./scripts/dom.js                            # page outline (title, headings, counts)
./scripts/dom.js 'form'                     # show matching elements' HTML
./scripts/dom.js --links                    # list all links
./scripts/dom.js --inputs                   # list all interactive elements
./scripts/dom.js --text 'main'              # extract visible text from a selector
```
